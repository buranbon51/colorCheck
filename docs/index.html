前書き

ここでは、 colorCheck と名付けたものについて説明します。

これを起動すると、以下の様なウインドウが表示します。

この colorCheck は、色を確認するために使うものです。

一番上の入力欄に、よくある RGB形式 の１６進数の６桁の文字を入力します。
( ３桁でも可能。 )

そして Enter キーを押すと、ウインドウ内の色が変更するので、それで色が確認できるというやつです。

そのついでに、下にある入力欄とスライドバーのコントロールに、１６進数と１０進数の数が反映される様になっています。

起動の仕方

この colorCheck を使うには、 AutoHotkey.exe が必要です。

この colorCheck の最初のフォルダ内に、 MainColorCheck.ahk というファイルがあるはずです。

そのファイルと同じフォルダに、 AutoHotkey.exe を配置してください。

後は、その AutoHotkey.exe を、 MainColorCheck.ahk のファイルがパラメータになるようにして実行すれば、 colorCheck が起動します。

例

ショートカットのファイルを用意して使うなら

リンク先
[colorCheckのフォルダ]\AutoHotkeyU32.exe  [colorCheckのフォルダ]\MainColorCheck.ahk

これをリンク先に指定しているショートカットを作れば、それから実行する事ができます。

補足

TextRunRun の アイテム から使える様にするなら、

リンク先
[colorCheckのフォルダ]\AutoHotkeyU32.exe
引数
[colorCheckのフォルダ]\MainColorCheck.ahk

こんなものを用意して使う様にする。

作業フォルダーは、無記入でも何を指定しても同じです。
MainColorCheck.ahk がある位置が作業フォルダーになる様になっています。

colorCheck の事あれこれ

この colorCheck はウインドウを見れば分かる通りのものです。

そして、あれが全てです。

なので、あれこれ書くのは返って読むのが煩わしい事になるのかもしれませんが、書ける事は書いておこうと思います。

終了の仕方

起動すると、ウインドウが表示します。
そしてこのウインドウがメインのウインドウです。

よって、ウインドウを閉じるとこの colorCheck は終了します。

Esc キーを押しても同じです。 colorCheck は終了します。

補足

私ブランボンが作ったものは、 Esc キーを押すとウインドウが閉じる様になっているものばかりだったりします。

ただ、 Esc キーを押した場合はソフトは終了せず、ウインドウが非表示になるだけのものが大半です。

そうなのですが、この colorCheck は Esc キーを押すと終了までする珍しいものだったりします。

一応、 AutoHotkey を使っているのでタスクトレイにアイコンが常駐するソフトだったりします。

ですが、タスクトレイにあるアイコンにはほとんど意味はありません。

終了する時はウインドウさえ閉じればよいです。

スライドバー

下の方に、３種類のスライドバーがあります。

R 、 G 、 B と、
赤、緑、青 の配色の括りに分かれています。

上が 0
下が 255
になり、その数字の色がウインドウに反映されます。

他の入力欄にも記入され、一番上の入力欄には RGB形式 の１６進数の６桁の文字が記入されます。

なのでこの色が RGB形式 で、どの文字列になるのかを確認するのに都合がいいです。

ボタン

いろいろボタンがありますが、文字の入力欄の一つ下のボタンがその入力欄用のボタンと思っていいです。

なので、入力欄に記入したら、一つ下のボタンを使うという事が分かればよいです。

ちなみに、右に On のボタンがありますが、これはウインドウに色を反映するかどうかという違いだけです。

それ以外は左側のボタンと同じです。
左側のボタンはその他の入力欄に文字を記入するだけですが、
On を押せば、その他の入力欄に記入しながら色をウインドウに反映します。
